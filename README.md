# RChecker 🚀

**RChecker** is a powerful and intuitive Roblox account utility designed to analyze combolists and identify accounts possessing specific avatar assets. It features a modern, dark-themed interface and multi-threaded processing for efficiency.

## ✨ Features
- **Multi-threaded**: Processes multiple accounts simultaneously using semaphores to prevent API bans.
- **Real-time Console**: Track the progress and errors live via the integrated log window.
- **Visual Results**: Displays the user's headshot and display name directly in the app.
- **Easy Export**: One-click button to copy successful matches to your clipboard.
- **Proxy Support**: Integrated with `link2proxy` for secure scanning.

## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/zZarby/RChecker.git
cd RChecker
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## 🚀 How to Use

1. Prepare a `.txt` file with your combolist in the format `Username:Password`.
2. Run the application:
```bash
python main.py
```
3. Click **Load Combolist** and select your file.
4. Hit **Start Checking** and watch the magic happen.
